# triangoli.v

fun game, I guess?


## build

1. get v from [vlang/v](https://github.com/vlang/v)
2. run `v -prod -skip-unused main.v`
3. hf with `main`


## build for dummies

```
git clone https://github.com/vlang/v.git
cd v
make
cd ..
git clone https://github.com/le0developer/triangoli.v.git
cd triangoli.v
../v/v -prod -skip-unused main.v
./main
```
